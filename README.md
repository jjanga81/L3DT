# L3DT
3차원 디지털 연합트윈 서비스 표출 기술 개발
- 경량 3D 변환 툴킷
3D CAD 모델을 SOFTHILLS Co., Ltd. (주)소프트힐스에서 제공하는 경량 파일 형식(*.HMF)의 파일을 생성합니다.
- HMF
생성된 HMF 파일은 디지털 연합트윈 3차원 시각화 엔진에서 가시화 할수 있습니다.
